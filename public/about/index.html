About SDFrame

About SDFrame, an independent open-source reference for software design frameworks maintained by Caldis.

What it is

SDFrame is an independent open-source reference site for software design frameworks. It collects 317 frameworks across 13 categories and makes them usable through a web UI, Markdown files, a static JSON API, and an AI skill package.

Maintainer

The project is maintained by Caldis. Source code and issue tracking are available at https://github.com/Caldis/frameworks. The site is public, read-only, and deployed as a static website.

Editorial stance

SDFrame is intended as an engineering reference. It favors concrete descriptions, source metadata, adoption examples, implementation steps, and explicit trade-offs over promotional claims. Framework entries should help users decide when a framework is useful and when it is not.
